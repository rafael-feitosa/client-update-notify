Um plugin simples para enviar notificações aos clientes após atualizações no site, via e-mail ou integração com algum WebHook.

== Description ==

Este plugin foi criado para desenvolvedores ou agências que fazem manutenção em sites WordPress e desejam informar seus clientes sobre atualizações realizadas.

**Principais recursos:**

- Envia notificações para clientes por e-mail diretamente do painel do WordPress.
- Suporte à integração com o Make (Integromat) através de Webhook.
- Permite configurar o assunto e o conteúdo do e-mail de forma personalizada.
- Interface simples e intuitiva em "Configurações > Notificações de manutenção".
- Possibilidade de disparar a notificação manualmente com apenas um clique.

Ideal para quem realiza manutenções recorrentes e quer manter o cliente informado de maneira profissional.

== Installation ==

1. Envie a pasta do plugin para o diretório `/wp-content/plugins/` ou instale diretamente via painel do WordPress.
2. Ative o plugin através do menu "Plugins".
3. Vá em **Configurações > Notificações de manutenção** para configurar o e-mail do cliente, template, assunto e URL do Webhook (opcional).
4. Salve as configurações e utilize o botão de disparo para enviar as notificações.

== Frequently Asked Questions ==

= O plugin envia notificações automaticamente? =  
Não. O envio é manual, feito pelo botão na página de configurações. Futuramente, funcionalidades automáticas podem ser adicionadas.

= É possível usar só o e-mail ou só o webhook? =  
Sim! Você pode ativar o envio por e-mail, usar só o webhook ou ambos, conforme preferir.
